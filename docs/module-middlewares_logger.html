Methods

(inner) LoggerMiddleware(req, res, next)

Este middleware registra la marca de tiempo, método HTTP, URL e IP para cada solicitud entrante. También mide y registra el tiempo de respuesta y el código de estado HTTP cuando la respuesta se completa.
Parameters:
Name Type Description
req Object Objeto de solicitud de Express
res Object Objeto de respuesta de Express
next function Función para pasar al siguiente middleware
